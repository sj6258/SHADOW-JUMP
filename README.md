# SHADOW-JUMP
A platformer game based on the popular Brackey's tutorial but with a twist. You can now teleport around the map and phase within blocks. Your goal? there are 25 coins around the level.. collect em all
WASD or arrow keys to move, Space to jump, E to teleport and have fun
