# SHADOW-JUMP
A platformer game based on the popular Brackey's tutorial but with a twist. You can now teleport around the map and phase within blocks. Your goal? there are 45 coins around the level.. collect em all
There are many levels in this game with more to come!
WASD or arrow keys to move, Space to jump, E to teleport and have fun
It also has a main meniu with a settings options to control volume and toggle fullscreen
